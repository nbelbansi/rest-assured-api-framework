# REST Assured API Automation Framework

## Overview
This project demonstrates API automation using REST Assured, TestNG, and Java.

## Features
- API Testing (GET, POST)
- Data-driven testing using TestNG DataProvider
- JSON Schema Validation
- Reusable framework (BaseTest, SpecBuilder)
- Logging and debugging

## Tech Stack
- Java
- REST Assured
- TestNG
- Maven

## How to Run
1. Clone repo
2. Run TestNG tests
